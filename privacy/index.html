Welche Daten verarbeitet QuickVoice?

QuickVoice ist eine reine On-Device-App ohne eigenes Backend. Der Entwickler hat keinen Zugriff auf deine Daten und sammelt keine Nutzungs-, Analyse- oder Werbedaten.

Daten, die dein Gerät nicht verlassen

Folgende Daten werden ausschließlich lokal auf deinem Gerät gespeichert und nicht an den Entwickler übertragen:

  • Dein OpenAI-API-Schlüssel – sicher im iOS-Schlüsselbund (Keychain).
  • Aufnahmeverlauf – transkribierte und formatierte Texte deiner Aufnahmen.
  • Benutzerwörter – eigene Begriffe zur besseren Spracherkennung.
  • Einstellungen – z. B. Ausgabesprache, Standard-Modus, Darstellung.

Diese Daten kannst du jederzeit in der App löschen. Beim Deinstallieren der App werden sie vollständig entfernt.

Übermittlung an OpenAI

Zur Transkription und Formatierung sendet QuickVoice die Audioaufnahme und den daraus entstehenden Text mit deinem persönlichen API-Schlüssel direkt an die Dienste von OpenAI (Whisper für die Transkription, GPT für die Formatierung). Diese Übertragung erfolgt über deinen eigenen OpenAI-Account.

Es findet keine Zwischenspeicherung auf einem Server des Entwicklers statt. Die Audiodatei wird nach der Transkription vom Gerät gelöscht. Welche Daten OpenAI verarbeitet und wie lange, richtet sich nach den Bedingungen von OpenAI: openai.com/policies/privacy-policy.

Abruf von Formatierungsvorlagen

QuickVoice lädt seine Formatierungs-Vorlagen (Text-Prompts und Modell-Liste) als statische Datei von GitHub (raw.githubusercontent.com). Bei diesem Abruf erhält GitHub technisch notwendige Verbindungsdaten wie deine IP-Adresse. Es werden dabei keine Inhalte deiner Aufnahmen übertragen. Hinweise: GitHub Privacy Statement.

Mikrofon

Der Mikrofonzugriff wird ausschließlich zum Aufnehmen genutzt, das du selbst startest. Die Berechtigung wird beim ersten Mal abgefragt und kann jederzeit in den iOS-Einstellungen widerrufen werden.

Kein Tracking, keine Werbung

QuickVoice enthält keine Analyse-Tools, kein Tracking, keine Werbung und keine Drittanbieter-SDKs zu diesen Zwecken.

Kinder

QuickVoice richtet sich nicht an Kinder und erhebt wissentlich keine Daten von Kindern.
